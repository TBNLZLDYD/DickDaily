# 撸管日历

一个科学的撸管提醒应用，帮助用户合理安排频率，维护身心健康。

## 功能特点

- 📅 日历界面，显示当前状态和建议
- ⚙️ 可自定义每周推荐频率
- 📋 详细的历史记录
- 🔔 智能提醒功能
- 🚀 支持开机自启动

## 科学依据

根据医学研究，适度的手淫对身体是有益的，建议频率为每周1-3次。过度或过少都可能对身体造成不良影响。

- 过度：可能导致疲劳、注意力不集中、性功能障碍等
- 过少：可能导致前列腺液淤积、性压力增加等

本应用基于个人体质差异，允许用户根据自身情况调整推荐频率。

## 界面说明

### 日历页面
- 显示当前日期和时间
- 显示上次记录时间和建议状态
- 提供"记录一次"和"重置历史"按钮

### 设置页面
- 可调整每周推荐频率
- 可开启或关闭提醒功能
- 可设置开机自启动

### 历史记录页面
- 显示所有历史记录，按时间倒序排列
- 包含日期和时间信息

## 数据存储

应用数据存储在用户主目录下的 `.dick_daily_data.json` 文件中，包含：
- 最后一次记录时间
- 推荐频率设置
- 提醒开关状态
- 自启动设置
- 历史记录

## 注意事项

1. 本应用仅供参考，具体频率应根据个人身体状况调整
2. 如有不适，请及时咨询专业医生
3. 保持健康的生活方式，包括合理饮食、充足睡眠和适量运动
